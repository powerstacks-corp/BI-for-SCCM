# BI for SCCM

Comprehensive Power BI reporting for Microsoft Configuration Manager (SCCM/ConfigMgr) by PowerStacks.

## Documentation

View the full documentation at [powerstacks-corp.github.io/BI-for-SCCM](https://powerstacks-corp.github.io/BI-for-SCCM/)

## About

BI for SCCM is a Power BI template app available on [Microsoft AppSource](https://appsource.microsoft.com) that provides comprehensive reporting for Microsoft Configuration Manager.
